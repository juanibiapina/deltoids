# Competitors

Diff and code review tools in the same space as deltoids.

- [hunk](https://github.com/modem-dev/hunk) — Review-first diff viewer with agent annotations, session control, TypeScript/Bun
- [lumen](https://github.com/jnsahaj/lumen) — Code review TUI with AI features (commit generation, PR support), Rust
- [delta](https://github.com/dandavison/delta) — Syntax-highlighting git pager, side-by-side, Rust
- [diff-so-fancy](https://github.com/so-fancy/diff-so-fancy) — Human-readable diff output, Perl
- [difftastic](https://github.com/Wilfred/difftastic) — Structural (AST-based) diffing, Rust
- [tuicr](https://github.com/agavra/tuicr) — Terminal UI for local code review, Rust
- [stage-cli](https://github.com/ReviewStage/stage-cli) — Local code review viewer that splits changes into small chapters, agent-friendly, TypeScript
