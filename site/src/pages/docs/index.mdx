# Install

## CLI

```bash
brew install juanibiapina/taps/deltoids
```

From source:

```bash
cargo install --git https://github.com/juanibiapina/deltoids deltoids-cli
```

A shell installer is also published on each
<a href={`${SITE.repo.url}/releases`}>GitHub release</a>.

## Agents

`deltoids edit`, `deltoids write`, `deltoids hashread`,
`deltoids hashedit`, and `deltoids traces` are the agent edit tools
and trace browser. Wire them into your agent:

### pi

```bash
pi install https://github.com/juanibiapina/deltoids
```

<Note>
  Set `DELTOIDS_EDIT_MODE=hash` before launching pi to swap the `edit`
  tool to a line-anchored backend (`deltoids hashedit`) and add a
  `hashread` tool for text-file reads. The default (`text` mode)
  keeps the classic `oldText`/`newText` `edit` schema. Mode is fixed
  for the lifetime of the pi session — changing it requires
  restarting pi.
</Note>

### Claude Code

```bash
claude plugin marketplace add juanibiapina/deltoids
claude plugin install deltoids@deltoids
```

<Note>
  Unlike the pi integration, Claude Code edits are recorded without
  per-edit summaries. Claude's `PostToolUse` hook does not expose
  one.
</Note>
